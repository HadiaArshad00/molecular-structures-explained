# Stereochemistry Practice Problems

## Problem Set 1: Fischer Projections

### Problem 1.1
Convert the following Fischer projection to a Newman projection viewing along the C2-C3 bond:
```
    CHO
    |
H—C—OH
    |
    CH₃
```

### Problem 1.2
Determine the R/S configuration:
```
      Br
      |
H₃C—C—OH
      |
      H
```

## Problem Set 2: Enantiomers vs Diastereomers

### Problem 2.1

- Answer: _______ (Enantiomers/Diastereomers)

### Problem 2.2

- How many stereoisomers exist?

## Problem Set 3: Cyclohexane Conformations

### Problem 3.1
Draw the chair conformation of cyclohexane with:
- Methyl group at C1 (equatorial)
- Chlorine at C3 (axial)

### Problem 3.2
Which conformation is more stable?
- (1R, 3S)-1-methyl-3-chlorocyclohexane with Me equatorial
- Same compound with Me axial

---

## Answer Key

**1.1**: Newman projection shows H and OH on adjacent carbons (staggered)

**1.2**: S configuration (Br-1, OH-2, CH₃-3)

**2.1**: Enantiomers

**2.2**: 4 stereoisomers total, 2 enantiomeric pairs, 1 meso compound

**3.1**: Check your drawing - equatorial is favored

**3.2**: Equatorial substitution is more stable
